# Overview

`api-dashboard` is a browser workspace for API request development inside Developer Dashboard.

It exists so API work can stay inside DD while still giving the user a practical request editor, collection manager, and response inspector.

The page includes:

- Collections and Workspace sections
- Postman-style collection import and export controls
- multiple request tabs
- request token value fields
- request credential presets
- response detail, body, and header panes
- saved Ajax worker definitions for collection bootstrap, collection save, collection delete, and request send

In practical terms, the skill helps the user keep API requests, tokens, auth settings, variables, and responses together in one place instead of scattering them across shell history and external tools.
